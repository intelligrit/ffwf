Fast Fuzzy Window Finder for macOS

Latest Version: 1.1.2

macOS 11.0 or later required

Features

  • Lightning-fast fuzzy search across all open windows
  • Global hotkey access (⌥⇧Space by default)
  • Menu bar integration - always accessible
  • Keyboard-driven navigation with arrow keys
  • Minimal resource usage - runs silently in the background
  • Smart scoring algorithm prioritizes relevant matches

Installation

  1. Download the DMG file above
  2. Open the downloaded FFWF-1.1.2.dmg file
  3. Drag FFWF.app to the Applications folder
  4. Launch FFWF from your Applications folder or Spotlight
  5. Grant Accessibility permission when prompted (required for window switching)

Usage

  • Click the magnifying glass icon in the menu bar, or
  • Press ⌥⇧Space (Option+Shift+Space) from anywhere
  • Type to filter windows using fuzzy search
  • Use ↑ and ↓ arrows to navigate
  • Press Enter to switch to the selected window
  • Press Esc to close without switching
